# Imsak Schedule
Scraper imsak schedule, valid for all regions of Indonesia.

### Usage
```js 
const imsakSchedule = require("./imsakSchedule.js");

imsakSchedule("jakarta", "12/03")
  .then(console.log)
  .catch(console.log)
  
imsakSchedule("jakarta")
  .then(console.log)
  .catch(console.log)
```
